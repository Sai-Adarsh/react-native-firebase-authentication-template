### Setup

```
git clone https://github.com/Sai-Adarsh/react-native-firebase-authentication-template.git
cd react-native-firebase-authentication-template
```

* Open App.js and replace with you firebase config
```
const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: ""
};
```
* Install dependencies
```
yarn install && npm install
react-native link
```
```
npm start
```
